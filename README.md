# Train Consist Management App
## Overview

The Train Consist Management App is a console-based Java application that simulates how railway systems manage a train’s consist (collection of bogies attached to an engine).

This project demonstrates the application of Core Java, Data Structures, Algorithms, and Exception Handling through real-world railway scenarios.

## Objective
Understand how data structures solve real-world problems
Implement step-by-step use cases (UC1–UC20)
Apply OOP, Collections, Streams, and Algorithms
Build strong problem-solving and system design skills
